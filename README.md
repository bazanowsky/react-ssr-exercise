# Apptension Exercise React SSR 


## Installation

```bash
yarn
```

## Running Dev Server 
### 2 Servers - client + server side
```bash
yarn run dev-all
```

###  server side only
```bash
yarn run dev
```

###  client side only
```bash
yarn run dev-client
```

## Open servers:

__Server side__ runs on: `localhost:3000`

__Client side__ runs on: `localhost:4000`

__Remember to force full refresh - Cmd+shift+R__
